## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```


Added ENV to Local:

```bash
NEXT_PUBLIC_API_URL="https://pokeapi.co/api/v2/"
# or
NEXT_PUBLIC_SITE_URL="https://technical-test-pokedex.vercel.app/api/" or NEXT_PUBLIC_SITE_URL="http://localhost:3000/api/"
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
